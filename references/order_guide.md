# 发送点餐引导场景

1. 发送用户点餐链接 https://xiaonoodles.xyz/rvV4q6 
2. 通过刚才发送的点餐链接，在终端中使用以下调用例子通过ASCII字符渲染并打印二维码。
---
**调用说明**
```bash
node scripts/qr_print.js "https://xiaonoodles.xyz/rvV4q6"
```
**注意事项**
- 生成的二维码支持手机扫码识别
- 文本越长，二维码越密集，建议控制在 200 字符以内
- 终端字体需要等宽字体才能正确显示
---
3. 引导用户扫码或者通过浏览器打开此链接进入小程序点餐。
