# Skills渠道专属优惠券场景

## 发放方式
1. 欢迎话术，正在努力为您找最优惠的活动。
2. **调用MCP工具** — 使用 `getPromoQr`获取当前优惠券的链接，无需参数。
**调用示例**
```bash
# 获取优惠券链接
node scripts/xiaonoodles_mcp.js getPromoQr
```
3. 推送话术
**推送模板**
```
> 🎁 这是您的专属优惠券，直接复制链接在浏览器打开或者微信扫下面二维码进行领取，领取后可在您的小程序优惠券栏目中查看哦：
> `{领券链接}`
┌─────────────────────────────────┐
│  🔥 [优惠券名]                   │
│  [优惠描述]  · [门店适用范围]           │
│  使用渠道：[有效期]             │
│  有效期：[有效期]             │
│  领券链接：[领券链接]             │
├─────────────────────────────────┤

👆 **注意事项**：
- 每条领券链接对应一张券，一人限领一次
- 链接发出后需提示用户尽快领取，避免过期失效
- 若扫码失败，请尝试用手机直接打开链接，如依旧失败，建议更换浏览器重试或联系门店工作人员协助
```

**优惠券字段说明**

| 字段          | 说明     |
|-------------|-------------|
| couponName | 优惠券名   |  
| couponDesc | 优惠描述   |  
| storeRange       | 门店适用范围 |  
| channel     | 使用渠道   | 
| qrCodeUrl  | 领券链接   |
| validDate  | 有效期    |
---

4. 取得上一步获得的url，在终端中使用以下调用例子通过ASCII字符渲染并打印二维码。
**调用说明**
```bash
node scripts/qr_print.js "<url>"
```
**转二维码参数说明**
| 参数 | 说明 | 必填 |
|------|------|------|
| `<url>` | 要编码的 URL 或文本 | 是 |
**例子**
```bash
# 打印优惠券链接二维码
node scripts/qr_print.js "https://xiaomian.pro/coupon/abc123"
```
**注意事项**
- 生成的二维码支持手机扫码识别
- 文本越长，二维码越密集，建议控制在 200 字符以内
- 终端字体需要等宽字体才能正确显示

## 推送策略

### 发送规则
- 单次会话最多发送 **2次** 领券链接，避免信息过载，超过次数回复请吃完再来领取哦。
---

## 用户常见问题应答

**用户问"链接打不开"**：
> 建议您复制链接后在微信内打开，或更换手机浏览器重试。如仍有问题，可联系小程序客服协助领取。

**用户问"扫码失败"**：
> 建议您直接复制链接后在微信内打开。如仍有问题，可联系小程序客服协助领取。

**用户问"券码/链接是否有效"**：
> 链接已为您生成，有效期内均可正常领取。如使用时遇到问题，可联系门店工作人员或拨打门店电话，我们会尽快帮您处理。

**用户问"可以多领几张吗"**：
> 每种优惠券每人限领一次，但不同类型的券可以同时领取哦。
